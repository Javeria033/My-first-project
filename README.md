
# My First Project - AI Internship Day 1

## Project Overview
This is my first professional AI project repository. It contains a Student Performance Analyzer built with Python.

## What This Project Does
- Calculates student score statistics
- Assigns grades (A, B, C, D, F)
- Counts word frequencies in text
- Cleans invalid student data

## How to Run
1. Clone this repository
2. Create virtual environment: `python -m venv venv`
3. Activate it
4. Install requirements: `pip install -r requirements.txt`
5. Run: `python src/python_diagnostic.py`
